About this App

This Version of the App is based on 7Pass, created by Tran Van Ngoc. You could find the original Source at https://github.com/7Pass/7Pass. Since there are no Updates to the original version, I have created my own Version, based on his Sourcecode. The primary goal for my new Version was Bugfixing, but in the meantime, I have added some more functionality to the App.

Currently I don't want to create my own App Developer Account for Publishing my Version to the Windows Phone Marketplace. So I have send my Sourcecode to Tran Van Ngoc, in hope, he will release a Update. But in the meantime, anyone with Visual Studio Express for Windows Phone could download my Source and Sideload this Version to the Phone.

Requirements

You need of course a Windows Phone. At this time, the App works on Windows Phone 8.1 (A Windows Phone 7 Version with nearby the same functionality is here).

Windows Phone 7

You need Visual Studio Express 2010 for Windows Phone and Zune installed. You could found this in the free Windows Phone 7.1 SDK. Additionaly, you should install the Update to the Windows Phone 7.8 SDK. You could found Zune at https://www.windowsphone.com/de-DE/How-to/wp7/zune-software. If this is all installed you can Download my Sourcecode. Load the Source in Visual Studio Express 2010 and start a "Build" of the whole Project. You get a XAP Package in the Debug Folder. Now you should Developer unlock your Phone (which is free for up to 5 Devices). Now you can "deploy" the XAP to your Phone. Be sure to Start Zune and connect the Phone to your PC via USB first.

Windows Phone 8.1

First you should the Download and install the Windows Phone 8.1 Developer Tools (Visual Studio Express 2013 for Windows). Now Download my Sourcecode for the WP8.1 Version. Load the Project in Visual Studio Express 2013 and start a "Build" of the whole Project. You get a XAP Package in the Debug Folder. After that, Developer unlock your Phone (which is free for up to 5 Devices). Now you can "deploy" the XAP to your Phone after you have connected the Phone via USB to your PC.

Whats new to this App?

Bugfixes:
  • Fixed Bug, which occurs by Sync a Database to any Cloudstorage resulting in every time having a conflicted copy of the Database
  • Fixed Bug, not encoding Password Strings correctly, so that copying this to the Clipboard and pasting it in the Target Window will result in wrong Password. You now a have a dedicated "Copy Password" Button on the Entry Screen.
  • Fixed Bug, not setting the correct timestamp after a Sync of every Database
  • Fixed Bug for a wrong arranged Contextmenu on the Database Page
New Features:
  • Optional AutoSync: Every Time the App is started, it looks now, if the Database in the Cloud is changed. If this is the Case, it gets the new Version. Optionally, AutoSync could be enabled only for WLAN Connections to save Traffic by your Carrier. AutoSync works only by opening the App. If you open the App from a pinned Database, AutoSync is temporarlly disabled
  • Optional Notification for Sync: you could get notified after the Sync via Toast and short Vibration
  • Optional Search in Password Fields: With this, you could search for a known Password, to check, if you use the same Password in other Entrys to
  • Complete Rework of the Layout on such many Places. Since this new App is based on 7Pass, it gets the Base of the old Layout and Design, but really all Screens are now updated and overhauled to gain a easier access to all the old and new functions of the App
  • Properties Page for every Database: With this you could not only rename the DB locally, you could also change the Cloud Credentials or the URL (Specially for WebDAV and WebServer Database Types)
  • Timestamp of the Actual DB on every Page
  • Smaller Contextmenu on DB Page, the Functions are now on a Properties Page.
  • Bigger Icons for Groups and Entrys, so it looks more like the official KeePass on Desktop
